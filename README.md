# FlyRank Backend AI Engineering – BE-01

## Project Overview

This project is a simple Express.js backend created as part of the FlyRank Backend AI Engineering Internship.

## Features

- GET `/`
- GET `/hello`

## API Endpoints

### GET /

Returns the server status.

Example response:

```json
{
  "status": "success",
  "message": "Backend server is running"
}
```

### GET /hello

Returns a greeting.

Example response:

```json
{
  "message": "Hello FlyRank!"
}
```

## Installation

```bash
npm install
```

## Run the Server

```bash
npm start
```

The server runs at:

```
http://localhost:3000
```
